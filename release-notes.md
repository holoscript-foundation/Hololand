## Quick Start

### With Brittney Desktop

1. Download [Brittney Desktop](https://github.com/brianonbased-dev/Hololand/releases)
2. The app will auto-download the model on first run

### With npm/pnpm

```bash
npx @hololand/brittney-models download v1-free
```

### Direct Download

```bash
curl -L -o brittney-v1-expert.gguf https://github.com/brianonbased-dev/Hololand/releases/download/brittney-v1.0.0/brittney-v1-expert.gguf
```
