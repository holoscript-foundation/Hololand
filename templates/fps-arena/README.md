# FPS Arena Template for HoloLand

Competitive multiplayer shooter arena with power-ups, round-based matches, leaderboards, and AI spectators/coaches.

## Features
- Round-based deathmatch / team modes
- Power-up spawns and weapon pickups
- Real-time leaderboards + ELO-style ranking (persisted via uAA2)
- AI Brittney commentators and post-match analysis
- Spectator mode with free-cam and replay ghosts
- Cross-world tournament integration

## How to Use
1. Create new world in Studio/HoloLand using this template.
2. Customize map geometry with HoloScript orbs + SDF.
3. Tune weapon/power-up traits.
4. Publish — monetization via entry fees or cosmetic marketplace.

## HoloScript Integration
Uses @template("fps_arena") with features for match orchestration, AI narration, and spatial audio zones.

See the base .hsplus for the declarative configuration.

Ties into the broader Growth Strategy: fast-to-publish competitive content for early user traction and monetization experiments.
