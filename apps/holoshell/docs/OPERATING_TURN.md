# HoloShell Operating Turn

The operating turn is the repeatable local loop for Brittney-in-HoloShell:

```powershell
pnpm run holoshell:operating-turn
```

It refreshes hardware reality, legacy windows, build custody, run custody,
legacy app absorption, readiness evidence, the operator brief, Brittney's
custody action, and the visual witness.

The turn is read-only by default. It records evidence and blocks process kills,
legacy app mutation, file deletion, and build termination behind preflight and
break-glass policies.
