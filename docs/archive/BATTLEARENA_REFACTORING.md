## Key Production Features

### 1. Damage System
```typescript
// Defense reduces incoming damage
actualDamage = Math.max(1, damage - defense/2)

// Examples:
- Fire Mage (attack: 15) vs Water (defense: 8)
  - Actual damage: 15 - 4 = 11
- Water (attack: 12) vs Fire (defense: 5)
  - Actual damage: 12 - 2.5 = 9.5
```

### 2. Projectile Physics
```typescript
// Calculate velocity towards target
const distance = √(dx² + dy² + dz²)
velocity = (target - position) / distance * speed

// Collision detection at ~2 unit range
if (distance < 2) → hit and damage
```

### 3. NPC AI Loop
```typescript
1. Find nearest enemy within 50 unit range
2. If distance < attackRange: fire projectile
3. If distance >= attackRange: move towards
4. Regenerate: mana += 10 * deltaTime
5. Update position every frame
```

### 4. Mana System
```typescript
- Cost per attack: 20 mana
- Regen rate: 10 mana per second
- Prevents constant attacks
- Creates combat pacing
```

### 5. Death Mechanics
```typescript
1. Health reaches 0 → set isAlive = false
2. Emit 'death:animation' event
3. Play 3-second animation
4. Remove from arena
5. Emit 'npc:destroyed' event
```

## Performance Characteristics

### Update Loop (60 FPS target)
```
Per Frame (16.67ms):
- Update projectiles: O(p) where p = projectile count
- Update NPC behavior: O(n) where n = NPC count
- Collision detection: O(n * p) worst case
```

```
Per NPC:

Total per NPC: ~2 KB

Per Projectile:
```

## Usage Example

```typescript
import { useBattleArena } from '@/hooks/useBattleArena'

function MyGameComponent() {
  const battle = useBattleArena()

  const startBattle = () => {
    // Spawn NPCs
    battle.spawnNPC({
      id: 'fire-1',
      name: 'Inferno',
      type: 'fire-mage',
      position: { x: 0, y: 0, z: 0 },
      maxHealth: 80
    })

    battle.spawnNPC({
      id: 'water-1',
      name: 'Aqua',
      type: 'water-elemental',
      position: { x: 30, y: 0, z: 0 },
      maxHealth: 100
    })

    // Start arena
    battle.startArena()
  }

  return (
    <div>
      <button onClick={startBattle}>Start Battle</button>
      
      {/* Display NPCs */}
      {battle.npcs.map(npc => (
        <div key={npc.id}>
          {npc.name}: {npc.health}/{npc.maxHealth}
        </div>
      ))}

      {/* Display events */}
      {battle.events.map((event, i) => (
        <div key={i}>{event.type}: {JSON.stringify(event.data)}</div>
      ))}
    </div>
  )
}
```
