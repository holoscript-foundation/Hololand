# Changelog

All notable changes to the HoloScript extension will be documented in this file.

## [1.0.0] - 2026-01-21

### Added
- Initial release
- Language support for `.holo` (HoloScript) files
- Language support for `.hsplus` (HoloScript Plus) files
- Syntax highlighting for both formats
- File icons with holographic gradient design
- Icon theme "HoloScript Icons"
- IntelliSense/auto-completion for:
  - Object types (cube, sphere, panel, etc.)
  - Properties (position, rotation, color, etc.)
  - Events (on_click, on_hover, etc.)
- `.hsplus` specific completions:
  - Networked objects
  - Physics constraints
  - Procedural generation
  - System imports

### Features by File Type

#### .holo (Standard HoloScript)
- Basic object definitions
- Materials and colors
- Simple animations
- UI panels and interactions

#### .hsplus (HoloScript Plus)
- All `.holo` features plus:
- Multiplayer networking keywords
- Physics constraint support
- Procedural generation syntax
- System import completions
